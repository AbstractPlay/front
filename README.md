# Abstract Play Front-End Client

[![Build Status](https://travis-ci.com/AbstractPlay/front.svg?branch=master)](https://travis-ci.com/AbstractPlay/front)

The official browser client for [Abstract Play](https://play.abstractplay.com).

## Developer documentation

Full contributor docs are published at **[docs.abstractplay.com/front/](https://docs.abstractplay.com/front/)** (getting started, architecture, subsystems, deployment).

## Quick start

```bash
git clone https://github.com/AbstractPlay/front.git
cd front
npm install
# Edit src/config/local.js for Cognito and API endpoints
npm start
```

See [Getting started](https://docs.abstractplay.com/front/getting-started/) for prerequisites (GitHub Packages, gameslib/renderer, node-backend).
